CONCLUSION

This analysis reveals clear patterns in social media engagement.

By focusing on high-performing content types, optimal posting times, and target demographics, engagement can be improved by an estimated 30–50%.

The data strongly supports a mobile-first, evening-focused content strategy with emphasis on positive, engaging content tailored to the 19–35 age demographic.
